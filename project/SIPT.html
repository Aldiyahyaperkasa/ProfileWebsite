Sistem Informasi Peminjaman Gedung

  • Tech: CodeIgniter 4, PHP, MySQL, JavaScript, HTML, CSS, Bootstrap
  • Libraries: Dompdf, SweetAlert
  • Github URL: Project Link

Produk yang dikembangkan adalah sebuah sistem informasi berbasis web yang memungkinkan untuk mengelola peminjaman tempat dan fasilitas secara lebih efisien dan terorganisir. Sistem ini menyediakan platform bagi pengguna untuk melakukan reservasi secara online, memudahkan administrasi dalam memantau status peminjaman, dan memberikan laporan analitik untuk pengambilan keputusan. Dengan adanya sistem ini, proses peminjaman yang sebelumnya manual dan memakan waktu dapat diotomatisasi, meningkatkan kemudahan akses dan transparansi.
